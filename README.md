# Aikido Kokikai Berlin ([kokikai.de](https://kokikai.de))

Website for Aikido Kokikai Berlin, the first and only Kokikai Aikido dojo in Europe.

Built with [Next.js](https://nextjs.org) 15, TypeScript, and Tailwind CSS. Deployed on [Vercel](https://vercel.com).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Create production build |
| `npm run start` | Serve production build locally |
| `npm run lint` | Run ESLint |

## Deployment

Push to `master` to auto-deploy via Vercel. Domains: `kokikai.de`, `www.kokikai.de`.
